# Simple Dataset Example

This example demonstrates the basic usage of CAST for managing a simple dataset.

## Overview

Shows how to:
- Create a basic manifest.json
- Use `cast.lib.mkDataset` to create a dataset derivation
- Access the dataset in a Nix flake

## Files (to be implemented in task 10)

- `flake.nix`: Flake that creates a simple dataset
- `manifest.json`: Example manifest for a small dataset
- `data/`: Sample data files

## Usage

```bash
nix build .#example-dataset
```
